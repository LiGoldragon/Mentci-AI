# Mentci-AI

Nix-and-Rust based AI daemon.

## Structure

- `flake.nix`: Root Nix entrypoint (Mutable Implementation).
- `src/`: Rust source code (Mutable Implementation).
- `Cargo.toml`: Rust manifest (Mutable Implementation).

## Usage

- `nix develop`: Enter the development environment.
- `nix build`: Build the package.
