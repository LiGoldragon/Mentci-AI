# Text Projection — Aski / Datalog Orientation

## Purpose
Provide an efficient, reviewable, and argument-friendly projection of protocol objects for agents and human auditors.

## Characteristics
- high semantic density,
- line-diff friendly,
- direct suitability for forum arguments and policy discussions,
- easy integration with existing repository review flow.

## Typical Use
- deliberation arguments,
- refusal feedback drafts,
- policy updates and lane mapping review,
- architecture evolution notes.

## Constraint
Text projection is not an informal note. It is an authoritative projection bound to a content hash and equivalent binary representation.
