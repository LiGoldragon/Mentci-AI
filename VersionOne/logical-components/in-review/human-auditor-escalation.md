# Human Auditor Escalation

## Review Focus
Standardize escalation path for unresolved deliberation cycles.

## Trigger
Second unresolved deliberation cycle on the same request lineage.

## Minimum Escalation Object
- request lineage hash,
- summarized argument graph,
- unresolved questions,
- blocked operation details.

## Open Review Topic
Resolution authority granularity after human input:
- direct approve/deny,
- bounded conditional approval,
- return-for-revision with mandatory constraints.
